Comecando o aprendizado como fazer um site basico em HTML.

O HTML (HyperText Markup Language) foi criado em 1991 pelo físico britânico Tim Berners-Lee, enquanto trabalhava no CERN (Organização Europeia para a Pesquisa Nuclear). Na época, Berners-Lee buscava uma forma de facilitar o compartilhamento e a organização de informações entre pesquisadores de diferentes partes do mundo. Para isso, ele desenvolveu: O HTML linguagem de marcação para estruturar documentos na web. O HTTP (HyperText Transfer Protocol) protocolo para transferência de dados. O primeiro navegador e servidor web ferramentas para acessar e disponibilizar páginas. O HTML permitiu criar documentos interligados por hiperlinks, formando a base da World Wide Web. Desde então, a linguagem evoluiu bastante, passando por diversas versões até chegar ao HTML5, que suporta recursos multimídia, gráficos e aplicações interativas sem a necessidade de plugins externos.
